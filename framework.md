# Choix du framework

J'ai décidé d'utiliser React dans un premier temps car c'est la technologie que j'utilise dans mon alternance.

Je pense que j'aurais pu utiliser n'importe quelle autre framework pour faire ce projet mais j'ai bien aimé apprendre à l'utiliser.

J'ai aussi utiliser React Router pour faire la gestion des routes au sein de l'application.
